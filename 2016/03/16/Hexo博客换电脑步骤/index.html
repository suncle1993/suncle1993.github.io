Hexo博客换电脑步骤

2016-03-16
Hexo博客
约 502 字 预计阅读 2 分钟

文章目录

hexo博客固然好，但是在多个电脑上怎么同步写博客呢？

搭建hexo环境:

要想在一个电脑上写博客，那肯定就得搭建node.js+hexo的环境。

首先，安装node.js

node.js的下载可以在node.js的中文网站下载，也可以使用下列百度云地址：

链接: https://pan.baidu.com/s/1kTVhcvd 密码: mqsy

然后安装git for windows

git for windows可以从百度的软件中心http://dlsw.baidu.com/sw-search-sp/soft/4e/30195/Git-2.7.2-32-bit_setup.1457942412.exe下载，也可以从github官网下载。还可以使用下面的百度云地址：

链接: https://pan.baidu.com/s/1eRtDECE 密码: cxh5

安装好nodej.js之后，打开git bash安装hexo，安装步骤如下：

1

npm install -g hexo

将之前备份的博客站点文件全部复制到本地的一个目录下。就可以使用下列命令操作了

1
2
3
4

hexo clean
hexo g
hexo s
hexo d

至于博客源文件可以用git管理，pull下来即可。

关于git管理博客源文件的方法参加另一篇blog：使用Github管理Hexo博客的源文件

附：

其实这一篇的主要目的只是为了放置nodejs和git的下载链接，因为官网是在是下载不了，速度慢的要屎了。
